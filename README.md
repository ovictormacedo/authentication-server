# authentication-server


This is a simple implementation of an authentication layer based on Oauth 2.

## Running

Use npm version >=7.6.0

```bash
cd src/
npm i
node app.js
```

## Running tests

Use npm version >=7.6.0

```bash
cd src/
npm test
```
